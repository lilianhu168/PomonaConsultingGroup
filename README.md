# Pomona Consulting Group Website

This is a static GitHub Pages-ready homepage for Pomona Consulting Group, combining the current PCG site content with a tighter Legora-inspired visual style.

## Edit Content

- Main page copy: `index.html`
- Styling and layout: `styles.css`
- Animations and interactions: `script.js`
- Videos/images you want to host directly: place them in `assets/`

## Hero Video

The hero uses `assets/pomona-hero.mp4`, a short self-hosted clip trimmed from the Pomona campus video, with `assets/pomona-hero-poster.jpg` as the poster image.

## Forms

The student interest and client interest forms currently create pre-filled email drafts to `pomonaconsulting@gmail.com`. To collect responses in a database or spreadsheet, replace the `data-mailto` setup with a backend such as Google Forms, Formspree, Netlify Forms, or a custom endpoint.

## Publish On GitHub Pages

1. Create a GitHub repository.
2. Upload these files to the repository root.
3. Go to repository `Settings` > `Pages`.
4. Set the source to deploy from the `main` branch root.
5. Your site will publish at `https://your-username.github.io/repository-name/`.
